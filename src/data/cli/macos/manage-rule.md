```
Add a rule to always manage the specified application

Usage: manage-rule <IDENTIFIER> <ID>

Arguments:
  <IDENTIFIER>
          Possible values:
          - exe:   Executable name
          - class: Role or subrole
          - title: Window title
          - path:  Executable path

  <ID>
          Identifier as a string

Options:
  -h, --help
          Print help (see a summary with '-h')
```
