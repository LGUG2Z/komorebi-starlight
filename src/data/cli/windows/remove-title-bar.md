Whitelist an application for title bar removal

Usage: remove-title-bar <IDENTIFIER> <ID>

Arguments:
  <IDENTIFIER>
          Possible values:
          - exe:   Executable name
          - class: Class
          - title: Window title
          - path:  Executable path

  <ID>
          Identifier as a string

Options:
  -h, --help
          Print help (see a summary with '-h')
