### margin {#margin}

<Badge text="optional" variant="note" /> [`SpacingKind`](#spacingkind)

Bar margin. Use one value for all sides or use a grouped margin for horizontal and/or
vertical definition which can each take a single value for a symmetric margin or two
values for each side, i.e.:
```json
"margin": {
    "horizontal": 10.0
}
```
or:
```json
"margin": {
    "vertical": [top, bottom]
}
```
You can also set individual margin on each side like this:
```json
"margin": {
    "top": 10.0,
    "bottom": 10.0,
    "left": 10.0,
    "right": 10.0
}
```

## Type Definitions

### AppConfig {#appconfig}

Application button configuration


<details open>
<summary>Properties (6)</summary>


#### command {#appconfig-command}

<Badge text="required" variant="success" /> `string`

Command to execute (e.g. path to the application or shell command).


#### name {#appconfig-name}

<Badge text="required" variant="success" /> `string`

Display name of the application.



#### display {#appconfig-display}

<Badge text="optional" variant="note" /> [`ApplicationsDisplayFormat`](#applicationsdisplayformat)

Display format for this application button (optional). Overrides global format if set.


#### enable {#appconfig-enable}

<Badge text="optional" variant="note" /> `boolean`

Whether to enable this application button (optional).
Inherits from the global `Applications` setting if omitted.


#### icon {#appconfig-icon}

<Badge text="optional" variant="note" /> `string`

Optional icon: a path to an image or a text-based glyph (e.g., from Nerd Fonts).
If not set, and if the `command` is a path to an executable, an icon might be extracted from it.
Note: glyphs require a compatible `font_family`.


#### show_command_on_hover {#appconfig-show_command_on_hover}

<Badge text="optional" variant="note" /> `boolean`

Whether to show the launch command on hover (optional).
Inherits from the global `Applications` setting if omitted.
